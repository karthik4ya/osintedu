# OSINT.EDU - Space Edition

A modern, interactive toolkit for Open Source Intelligence (OSINT) tools, built with React and Vite.

## Features

- Orbital viewer for visualizing data
- Search bar for querying tools
- Tool cards with simulated command execution
- AI-powered responses using Google Generative AI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/karthik4ya/osintedu.git
   cd osintedu
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   - Create a `.env` file in the root directory.
   - Add your Google Generative AI API key:
     ```
     API_KEY=your_api_key_here
     ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173` (or the configured URL).

## Usage

- Use the search bar to find OSINT tools.
- Click on tool cards to simulate command execution.
- For file uploads, select a file and run the tool.
